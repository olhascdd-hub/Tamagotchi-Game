# Tamagotchi-Game
Take care of your virtual pet!
This is a small-scale Tamagotchi-style application developed using C# and Windows Forms.
The user can create and take care of a virtual pet by performing different actions.

Features
1.Create a pet (Duck, Bunny, Kitten)
2.Action system:
- Feed
- Play
- Sleep
- Clean
- Treat
- Wait
- Rock (with sound)
3.Stat system:
-Hunger
-Happiness
-Energy
-Cleanliness
4.Colour-coded status indicators
5.Warning system
6.Game over system
7.Restart functionality
8.Dynamic pet images

Technologies Used:
C#
Windows Forms
System.Drawing
WMPLib

How to Run:
Open the solution in Visual Studio
Build the project
Run the application
